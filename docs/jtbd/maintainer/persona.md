# Maintainer

## Who
A developer contributing to or maintaining the addressr-mcp open-source repository.

## Context Constraints
- Works with pure ESM Node.js codebase
- Must keep the proxy thin and secure
- Follows changesets-based release workflow

## Pain Points
- Keeping proxy behaviour aligned with upstream Addressr API changes
- Managing releases without leaking secrets
- Ensuring MCP protocol compliance
