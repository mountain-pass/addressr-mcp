# Developer

## Who
A software developer building applications that need Australian address validation or autocomplete functionality.

## Context Constraints
- Uses MCP-compatible clients (Claude Desktop, Claude Code, etc.)
- Needs accurate, up-to-date Australian address data
- Wants simple tool-based integration without managing API keys directly

## Pain Points
- Australian address formats are complex and inconsistent
- Maintaining address data locally is burdensome
- Direct API integration requires managing authentication and endpoints
